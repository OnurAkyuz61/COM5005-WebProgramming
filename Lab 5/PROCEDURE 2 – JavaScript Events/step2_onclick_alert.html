Step 2 - Button with JavaScript Event

Note: Now when the button is clicked, "Hello!" Let's do it with JavaScript.

Here, thanks to the HTML feature that we can use as onclick="", we made the codes we wrote in JavaScript run when the button is clicked.

HTML Events Table:

Event Description
onchange Raised when the HTML element is changed.
onclick Raised when the user clicks on the HTML object.
onmouseover Raised when the user hovers over the HTML object.
onmouseout Raised when the user leaves the HTML object.
onkeydown It is triggered when the user presses the keyboard key.
onload It is triggered when the browser has finished loading the page.
